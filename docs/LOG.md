# 執行日誌 (LOG)

> **機器寫的**。orchestrator 每次呼叫 CLI、每次額度換手、每個產出都會自動追加在這裡。
> 這是換手後接力的「交接棒」之一 —— 下一家可以從這裡看到前面發生過什麼。

格式：`時間` **provider** → 狀態 · 說明

| 狀態 | 意思 |
|---|---|
| `ok` | 成功完成，附產出摘要 |
| `limit` | 額度耗盡，附冷卻到期時間 |
| `auth` | 認證/登入問題，需人工處理 |
| `error` | 其他錯誤或逾時 |
